# gbcompo25

gbcompo25 was a game jam run from May 31st to August 31st 2025, hosted on https://itch.io/jam/gbcompo25

This repository keeps an archive of the submitted entries.

More information at https://gbdev.io/gbcompo25

Currently sources for the open-source entries are archived as individual repos here: https://github.com/gbcompo25/
